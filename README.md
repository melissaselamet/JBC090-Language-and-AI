# JBC090-Language-and-AI
## Data Preparation
To run this code, the file "political_leaning.csv" should be present in the working directory.
To be able to run the code without running into errors, you either also need "E_original.csv" in the working directory, or un-comment the "to_csv" and "read_csv" lines.
Lastly, a few more encounters of such lines can be found, than can be un-commented or left commented.
Such lines are indicated by a comment that clarifies this.
## Model Running
To run this code, the files "N_original.csv", "E_original.csv", and "E_cleaned.csv" should be in the working directory.
## Result Analysis
To run this code, the files "results.csv", "results2.csv", and "results3.csv" should be in the working directory.
